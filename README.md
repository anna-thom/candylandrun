# candylandrun
CS 4361| Christmas Wonderland: First person platfomer over candy/christmas-like biome
Ensure you have the Godot game engine installed.
Reclone the github repository and reimport the game before finally pressing play "Run Current Scene". 
How to play:
Controls - WASD, Space to jump, 0 for no boot equipment, 1 for jump boots, 2 for sprinting boots
W - go forward
A - go left
S - go backward
D - go right
